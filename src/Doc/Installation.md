# 安装

## 使用 npm 安装

```bash
npm install @zhenryx/zhenryx-react-native-ui
```

## 快速开始

安装完成后，在项目中导入组件：

```tsx
import { Button } from '@zhenryx/zhenryx-react-native-ui';

function App() {
  return (
    <Button>点击我</Button>
  );
}
```

## 注意事项

- 确保你的 React Native 版本符合要求
- 某些组件可能需要额外的依赖，请查看具体组件的文档
